实现方案

  • 纯前端改动，不改后端 API
  • 遍历 chunks，比较相邻 chunk 的 parent_content，相同的归为一组
  • 每组分配一个颜色标签（蓝/绿/黄轮换），显示"父块 A/B/C"
  • 同组间卡片头尾相连（取消圆角），左侧加色条连接线
  • 同组用统一浅色背景，方便视觉区分
